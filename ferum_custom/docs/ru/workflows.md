# Рабочие процессы (Workflow)

- Issue Workflow: Open → In Progress → Resolved → Closed/Cancelled.
- Timesheet Workflow: Draft → Submitted → Approved → Cancelled/Rejected.
- Project Workflow: Planned → Active → Completed/Cancelled.
- Invoice Workflow: Draft → Sent → Paid/Cancelled.
