# Рабочие процессы (Workflow)

- Service Request Workflow: Open → In Progress → Completed → Closed/Cancelled.
- Service Report Workflow: Draft → Submitted → Approved → Archived/Cancelled.
- Service Project Workflow: Planned → Active → Completed/Cancelled.
- Invoice Workflow: Draft → Sent → Paid/Cancelled.

Фикстуры для Workflow/Action/State включены в репозиторий: apps/ferum_custom/ferum_custom/hooks.py (раздел fixtures).
